# Author Context Archive

*Structured biographical context for authentic content generation*

**Last updated:** 2026-01-13

---

## Structure

```
context/
├── author.md           # Overview and quick reference
├── README.md           # This file
├── timeline/           # Chronological life events
│   ├── 1990s.md        # Early career, formative years
│   ├── 2000s.md        # Middle years
│   ├── 2010s.md        # Pre-coding era
│   └── 2020s.md        # Coding journey
├── people/             # Key relationships
│   ├── family.md       # Immediate and extended family
│   ├── mentors.md      # Teachers, guides, influences
│   └── collaborators.md # Professional relationships
├── places/             # Geographic context
│   ├── origins.md      # Where from originally
│   └── hungary.md      # Current life in Hungary
└── themes/             # Cross-cutting patterns
    ├── spiritual.md    # Buddhist practice, Aka Dua, etc.
    ├── learning.md     # How learning happens
    └── career.md       # Professional arc
```

## Usage

**Before content generation**, load:
1. `author.md` - Always (quick reference)
2. Relevant category files based on article topic

## Update Protocol

- Add new findings to appropriate category file
- Update `author.md` overview when significant facts confirmed
- Note sources (email thread ID, calendar date, etc.)
- Mark confidence level (confirmed/inferred/speculative)

## Privacy Boundaries

- Public figures and professional contacts: Named
- Private relationships: Use discretion
- Financial/health: Aggregate patterns only
- No raw email content in files
